# Outreach Send Checklist

Before sending each email:
- [ ] Recipient name/team is correct
- [ ] Subject line matches target company
- [ ] Body has no placeholder text
- [ ] CTA asks for 15-minute call
- [ ] Signature includes `techgeniexyz@gmail.com`
- [ ] Sent email logged in tracker

After sending:
- [ ] Set follow-up #1 reminder (48h)
- [ ] Set follow-up #2 reminder (72h after follow-up #1)
- [ ] Update pipeline status
