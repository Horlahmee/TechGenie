# TechGenie Company OS Starter Pack

This is the operating system for running TechGenie as a CEO-led multi-agent company.

## Core Principles
- CEO sets direction and approves major decisions.
- COO (Super Genie) plans, delegates, QA-checks, and reports.
- Specialists execute narrow tasks quickly.
- Every task has an owner, deadline, and success criteria.

## Folder Map
- `roles/` — role definitions and authority boundaries
- `sops/` — standard operating procedures
- `templates/` — reusable templates
- `cadence/` — daily/weekly/monthly ritual docs
- `projects/` — active initiatives
- `reports/` — status, KPI, and postmortem outputs

## Launch Checklist
1. Confirm role assignments in `roles/ROLE-CHART.md`
2. Use `templates/PROJECT-BRIEF.md` to start first project
3. Open first daily standup using `cadence/DAILY-STANDUP.md`
4. Track execution using `templates/TASK-BOARD.md`
5. Send CEO summary using `templates/CEO-BRIEF.md`
